# 2. 수익 구조 및 AdSense 전략 (Revenue Strategy)

## 2.1 Ad Placement Logic
Goal: Maximize Viewability & CTR without triggering "Policy Violation".

### Layout Level
1.  **Global Header Below**: 728x90 Leaderboard (Desktop) / 320x100 (Mobile).
    - Fixed visibility on initial load.
2.  **Sidebar (Desktop)**: 300x600 Sticky Unit.
    - Moves with scroll to maintain visibility.

### Page Level
3.  **Content Injector (Core)**:
    - **Post-H1**: 300x250 Rectangle.
    - **Paragraph 3**: In-article native ad.
    - **Calculation Result**: **Critical Zone**.
        - *Immediately above* result: "Sponsored Link" style.
        - *Immediately below* result: 300x250 or Responsive.

### Interactive Elements
4.  **Calculator Interstitial**:
    - When "Calculate" is clicked -> Show modal or loading state with Ad -> Show Result.
    - (Must ensure compliance: do not encourage accidental clicks).

## 2.2 High RPM Keywords
Focus content generation on:
- "Dividend Tax Exemption" (배당소득세 계산)
- "Roth IRA Calculator" (ISA 계좌 혜택)
- "Inheritance Tax" (상속세 자동 계산)

## 2.3 User Flow for Revenue
Landing (SEO) -> Calculator Use -> Result Page (Ad Exposure) -> "Related Stocks" (Click) -> Stock Hub (Ad Exposure) -> News (Ad Exposure).
**Minimum 3 PV per session target.**
